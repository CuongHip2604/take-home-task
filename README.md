# Time Series Download

## Installation

1. Clone the repository:

```bash
git clone https://github.com/MichaelNguyenAdamo/take-home-task.git
```

2. Navigate to the project directory:

```bash
cd take-home-task
```

3. Install dependencies:

```bash
npm install
```

4. Start Json Server

```bash
npm run json-server
```

5. Start project

```bash
npm run dev
```
